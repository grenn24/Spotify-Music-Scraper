Welcome to Spotify Music Scraper

This is the homepage of the Spotify Music Scraper application.

You can perform various actions using the endpoints provided by the application:

  • Get all songs
  • Get YouTube URL for all songs
  • Get YouTube URL for a specific song
  • Generate insights based on the songs
  • Get Spotify API access token
  • Delete all songs
  • Delete a specific song

You can use the provided links to interact with the application.

For more details, please refer to the API documentation.

For any issues or inquiries, please contact the administrator.
